# 📖 English — Error Detection (Basics) | Day 3

> Error Detection = sentence ko parts me toda jaata hai, batana hota hai kis part me galti hai (ya "No error").
> Yeh Prelims ka **high-frequency** topic hai. Galti zyadatar in 5 jagah hoti hai — wahi pehle check karo.

---

## 1. Galti dhoondhne ka Order (yeh sequence follow karo)

1. **Subject-Verb Agreement** — singular subject → singular verb?
2. **Tense** — sentence ke baaki hisse se tense match karta hai?
3. **Article (a/an/the)** — sahi article laga hai?
4. **Preposition** — sahi preposition (in/on/at/to/for/with)?
5. **Noun/Pronoun** — singular/plural, he/him sahi?

> Agar in 5 me kuch galat nahi → answer **"No error"** ho sakta hai. (Har sentence me galti zaroori nahi.)

---

## 2. Common Galtiyan (rules + example)

**(a) Subject-Verb Agreement**
- ❌ One of my friends *are* a doctor. → ✅ *is* (One of... = singular)

**(b) Tense mix**
- ❌ He said that he *will* come. → ✅ *would* (past ke baad past)

**(c) Article**
- ❌ He is *an* European. → ✅ *a* (European = "yoo" sound)

**(d) Preposition (yaad rakho yeh common pairs)**
- married **to**, good **at**, afraid **of**, listen **to**, depend **on**, reason **for**, capable **of**.
- ❌ She is married *with* him. → ✅ married **to**

**(e) Double negative / extra word**
- ❌ I did not see *nobody*. → ✅ I did not see **anybody**.

**(f) Pronoun**
- ❌ Each boy must bring *their* book. → ✅ **his** (Each = singular)

---

## 3. Mini Rule Box — Preposition Combos (ratta)

| Word | Preposition |
|---|---|
| good / bad / expert | **at** |
| afraid / fond / capable / sure | **of** |
| depend / rely / insist | **on** |
| married / listen / superior / prefer | **to** |
| reason / famous / responsible | **for** |
| angry (kisi cheez par) | **at/about** |

## 4. ✍️ Practice — Error Detection (part dhoondho)

1. (A) The list of items / (B) are / (C) on the table. / (D) No error
2. (A) He is junior / (B) than me / (C) by two years. / (D) No error
3. (A) She is good / (B) in / (C) mathematics. / (D) No error
4. (A) Neither of the two boys / (B) have / (C) come today. / (D) No error
5. (A) He has been working / (B) here / (C) since five years. / (D) No error
6. (A) I prefer tea / (B) than / (C) coffee. / (D) No error
7. (A) One of the student / (B) was / (C) absent. / (D) No error
8. (A) The Sun / (B) rises / (C) in the East. / (D) No error
9. (A) He did not / (B) do / (C) nothing. / (D) No error
10. (A) She is senior / (B) to / (C) me. / (D) No error

---

## ✅ Answers

1. **(B)** are → **is** (The list = singular)
2. **(B)** than → **to** (junior **to**)
3. **(B)** in → **at** (good **at**)
4. **(B)** have → **has** (Neither = singular)
5. **(C)** since → **for** (since = point of time; for = duration)
6. **(B)** than → **to** (prefer X **to** Y)
7. **(A)** student → **students** (One of the + plural)
8. **(D)** No error ✅ (sentence sahi hai)
9. **(C)** nothing → **anything** (double negative galat)
10. **(D)** No error ✅ (senior **to** sahi)
