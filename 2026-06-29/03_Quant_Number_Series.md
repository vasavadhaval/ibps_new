# 🟡 Quant — Number Series | Day 6

> Number Series = numbers ki ek line di hoti hai jo ek **pattern** follow karti hai.
> Poocha jaata hai: **missing number** ya **wrong number** dhoondho. Easy + scoring (5 marks).

---

## 1. Common Patterns (yeh dhyaan se dekho)

| Pattern | Example |
|---|---|
| **+ / − constant** | 2, 5, 8, 11... (+3) |
| **× / ÷ constant** | 3, 6, 12, 24... (×2) |
| **Square** | 1, 4, 9, 16, 25... (n²) |
| **Cube** | 1, 8, 27, 64... (n³) |
| **+ increasing diff** | 2, 3, 6, 11, 18... (+1,+3,+5,+7) |
| **Prime numbers** | 2, 3, 5, 7, 11, 13... |
| **×n + / − k** | 2, 5, 11, 23... (×2 +1) |
| **Alternate series** | do alag patterns mile hue |

---

## 2. Sabse Useful Trick

> **Consecutive numbers ke beech ka DIFFERENCE nikaalo.** Agar difference constant → +/− series. Agar difference khud ek pattern bana raha (1,3,5...) → increasing series. Agar ratio constant (×2) → multiply series.

Squares/cubes yaad hon (Day 1/2 waale) toh `1,4,9,16` / `1,8,27` turant pakad loge.

---

## 3. Solved Examples

**Example 1 (missing):** `3, 6, 12, 24, ?`
- Ratio ×2 → 24×2 = **48** ✅

**Example 2 (missing):** `2, 3, 6, 11, 18, ?`
- Diff: +1, +3, +5, +7 → next +9 → 18+9 = **27** ✅

**Example 3 (square based):** `1, 4, 9, 16, ?`
- n²: 1²,2²,3²,4²... → 5² = **25** ✅

**Example 4 (×n + k):** `2, 5, 11, 23, ?`
- ×2 +1 har baar → 23×2+1 = **47** ✅

**Example 5 (wrong number):** `4, 8, 16, 30, 64`
- ×2 hona chahiye: 4,8,16,**32**,64. → galat number = **30** (32 hona chahiye) ✅

---

## 4. Solve karne ka Tarika (steps)

1. Pehle **difference** nikaalo do-do numbers ka.
2. Difference constant? → +/− series. Constant nahi? → difference ka pattern dekho.
3. Multiply/divide ka ratio check karo (×2, ×3...).
4. Squares/cubes/primes se match ho raha kya?
5. "Wrong number" me: pattern ko aage badha ke dekho kaun fit nahi hota.

---

## 5. ✍️ Practice Questions (missing/wrong number)

1. `5, 10, 20, 40, ?`
2. `1, 4, 9, 16, 25, ?`
3. `2, 6, 12, 20, 30, ?`
4. `3, 7, 15, 31, ?`
5. `100, 50, 25, ?`
6. `1, 8, 27, 64, ?`
7. `7, 14, 28, 56, ?`
8. (wrong) `2, 5, 10, 17, 26, 38` — galat kaun?
9. `4, 9, 16, 25, 36, ?`
10. `3, 6, 11, 18, 27, ?`

---

## ✅ Answers (solve karne ke BAAD)

1. ×2 → 40×2 = **80**
2. n² → 6² = **36**
3. diff +4,+6,+8,+10 → 30+12 = **42** (ya n²+n: 30+ ... → **42**)
4. ×2 +1 → 31×2+1 = **63**
5. ÷2 → 25÷2 = **12.5**
6. n³ → 5³ = **125**
7. ×2 → 56×2 = **112**
8. pattern +3,+5,+7,+9,+11 → 2,5,10,17,26,**37**. Diya 38 → galat = **38**
9. n² (from 2²): 4,9,16,25,36 → 7² = **49**
10. diff +3,+5,+7,+9,+11 → 27+11 = **38**

---

> **Aaj ka takeaway:** Number Series me sabse pehle **difference** nikaalo. Constant diff = add series, badhta diff = square/increasing, ratio = multiply. Squares (1-20) aur cubes (1-12) yaad hon toh aadhe questions seconds me. Roz 10 series karo.
