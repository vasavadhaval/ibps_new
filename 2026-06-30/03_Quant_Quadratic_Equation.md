# 🟡 Quant — Quadratic Equation | Day 7

> Quadratic Equation IBPS Prelims ka fixed-pattern topic hai. Usually do equations di hoti hain: ek x ke liye, ek y ke liye.
> Roots nikaalke compare karna hota hai: x > y, x < y, x ≥ y, x ≤ y, x = y, ya relation cannot be determined.

---

## 1. Basic Form

```
ax² + bx + c = 0
```

Prelims me mostly factorization se solve hota hai.

Example:
```
x² - 5x + 6 = 0
```

Socho: kaunse 2 numbers ka product 6 aur sum 5? → 2 and 3.

```
x² - 5x + 6 = 0
(x - 2)(x - 3) = 0
x = 2, 3
```

---

## 2. Sign Shortcut

| Equation type | Roots ka sign |
|---|---|
| x² - (sum)x + product = 0 | dono roots positive |
| x² + (sum)x + product = 0 | dono roots negative |
| x² + bx - c = 0 | ek positive, ek negative |
| x² - bx - c = 0 | ek positive, ek negative |

---

## 3. Comparison Symbols

After roots:
- Agar x ke dono roots y ke dono roots se bade → **x > y**
- Agar x ke dono roots y ke dono roots se chhote → **x < y**
- Agar kuch bade, kuch chhote → **relation cannot be determined**
- Agar same roots ho sakte hain → **x = y** ya mixed relation check karo

---

## 4. Solved Examples

**Example 1:**
```
x² - 5x + 6 = 0  → x = 2, 3
y² - 9y + 20 = 0 → y = 4, 5
```
x roots 2,3 and y roots 4,5. Har x, y se chhota → **x < y** ✅

**Example 2:**
```
x² - 7x + 12 = 0 → x = 3,4
y² - 5y + 6 = 0  → y = 2,3
```
x = 3,4 and y = 2,3. 3 equals 3, 4 greater than both → **x ≥ y** ✅

**Example 3:**
```
x² - 5x + 6 = 0 → x = 2,3
y² - 5y + 4 = 0 → y = 1,4
```
x=2,3; y=1,4. Kabhi x>y, kabhi x<y → **relation cannot be determined** ✅

---

## 5. Solve Karne Ka Tarika

1. Dono equations ke roots nikalo.
2. Roots ko ascending order me likho.
3. Har x ko har y se compare karo.
4. Agar full clear relation ho tabhi answer do.
5. Mixed ho toh **cannot be determined**.

---

## 6. ✍️ Practice Questions

Compare x and y:

1. `x² - 5x + 6 = 0`; `y² - 9y + 20 = 0`
2. `x² - 7x + 12 = 0`; `y² - 5y + 6 = 0`
3. `x² - 5x + 6 = 0`; `y² - 5y + 4 = 0`
4. `x² - 3x + 2 = 0`; `y² - 7y + 12 = 0`
5. `x² + 5x + 6 = 0`; `y² - 5y + 6 = 0`
6. `x² - 8x + 15 = 0`; `y² - 8y + 16 = 0`
7. `x² - 6x + 9 = 0`; `y² - 4y + 4 = 0`
8. `x² + 7x + 12 = 0`; `y² + 5y + 6 = 0`
9. `x² - 10x + 21 = 0`; `y² - 6y + 8 = 0`
10. `x² - 4x + 4 = 0`; `y² - 4y + 4 = 0`

---

## ✅ Answers

1. x=2,3; y=4,5 → **x < y**
2. x=3,4; y=2,3 → **x ≥ y**
3. x=2,3; y=1,4 → **cannot be determined**
4. x=1,2; y=3,4 → **x < y**
5. x=-2,-3; y=2,3 → **x < y**
6. x=3,5; y=4,4 → mixed (3<4, 5>4) → **cannot be determined**
7. x=3,3; y=2,2 → **x > y**
8. x=-3,-4; y=-2,-3 → **x ≤ y**
9. x=3,7; y=2,4 → mixed (3>2 but 3<4; 7>both) → **cannot be determined**
10. x=2,2; y=2,2 → **x = y**

---

> **Aaj ka takeaway:** Quadratic me speed factorization se aati hai. Roots nikaalo, sorted likho, phir compare karo. Mixed relation dikhe toh force mat karo — answer cannot be determined hota hai.
