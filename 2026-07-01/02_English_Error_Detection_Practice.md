# 📖 English — Error Detection Practice | Day 8

> Week 2 English ka focus: Error Detection ko **practice** se strong karna.
> Order yaad rakho: **SVA → Tense → Article → Preposition → Pronoun**. "No error" bhi valid hai.

---

## 1. Quick Revision — Check Order

1. **Subject-Verb Agreement** — singular subject → singular verb?
2. **Tense** — sentence ke baaki hisse se match?
3. **Article** — a/an/the sahi?
4. **Preposition** — good at, married to, afraid of, depend on...
5. **Pronoun** — he/him, each/every singular?

---

## 2. Common Preposition Pairs (ratta)

| Word | Preposition |
|---|---|
| good / bad / expert | **at** |
| afraid / fond / capable / sure | **of** |
| depend / rely / insist | **on** |
| married / listen / superior / prefer | **to** |
| reason / famous / responsible | **for** |
| junior / senior | **to** (than ❌) |

## 3. ✍️ Practice — Error Detection (part dhoondho)

1. (A) The number of students / (B) are / (C) increasing. / (D) No error
2. (A) He is good / (B) in / (C) mathematics. / (D) No error
3. (A) She has been working / (B) here / (C) since five years. / (D) No error
4. (A) Neither of the two answers / (B) are / (C) correct. / (D) No error
5. (A) I prefer tea / (B) than / (C) coffee. / (D) No error
6. (A) The news / (B) are / (C) very shocking. / (D) No error
7. (A) He is junior / (B) than / (C) me by two years. / (D) No error
8. (A) Each of the players / (B) were / (C) given a medal. / (D) No error
9. (A) She is afraid / (B) from / (C) dogs. / (D) No error
10. (A) One of my friends / (B) are / (C) a doctor. / (D) No error
11. (A) He did not / (B) see / (C) nobody. / (D) No error
12. (A) The Sun / (B) rises / (C) in the East. / (D) No error
13. (A) She is married / (B) with / (C) a teacher. / (D) No error
14. (A) The quality of apples / (B) are / (C) excellent. / (D) No error
15. (A) He is an / (B) European / (C) citizen. / (D) No error

---

## ✅ Answers

1. **(B)** are → **is** (The number of = singular)
2. **(B)** in → **at** (good **at**)
3. **(C)** since → **for** (since = point; for = duration)
4. **(B)** are → **is** (Neither = singular)
5. **(B)** than → **to** (prefer X **to** Y)
6. **(B)** are → **is** (news = singular)
7. **(B)** than → **to** (junior **to**)
8. **(B)** were → **was** (Each = singular)
9. **(B)** from → **of** (afraid **of**)
10. **(B)** are → **is** (One of... = singular)
11. **(C)** nobody → **anybody** (double negative)
12. **(D)** No error ✅
13. **(B)** with → **to** (married **to**)
14. **(B)** are → **is** (subject = quality)
15. **(A)** an → **a** (European = yoo sound)
