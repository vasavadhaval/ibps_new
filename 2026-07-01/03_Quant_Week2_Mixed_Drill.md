## 1. 60-Second Recap

**Number Series:**
- Pehle **difference** nikaalo. Constant → +/−. Badhta diff → square/increasing. Ratio → ×/÷.
- Squares 1-20, cubes 1-12 yaad hon toh speed double.

**Quadratic Equation:**
- Roots nikaalo (factorization). Sorted likho. Compare x vs y.
- Mixed relation → **cannot be determined**.

## 2. ✍️ Mixed Practice (timer: 15 min)

**Set A — Number Series (Q1-5):**
1. `5, 10, 20, 40, ?`
2. `2, 6, 12, 20, 30, ?`
3. `1, 4, 9, 16, 25, ?`
4. (wrong) `2, 5, 10, 17, 26, 38` — galat number?
5. `3, 7, 15, 31, ?`

**Set B — Quadratic Compare x and y (Q6-10):**
6. `x² - 5x + 6 = 0`; `y² - 9y + 20 = 0`
7. `x² - 7x + 12 = 0`; `y² - 5y + 6 = 0`
8. `x² - 4x + 4 = 0`; `y² - 4y + 4 = 0`
9. `x² - 5x + 6 = 0`; `y² - 5y + 4 = 0`
10. `x² - 3x + 2 = 0`; `y² - 7y + 12 = 0`

---

## ✅ Answers (solve karne ke BAAD)

**Set A:**
1. ×2 → 40×2 = **80**
2. diff +4,+6,+8,+10 → 30+12 = **42**
3. n² → 6² = **36**
4. pattern +3,+5,+7,+9,+11 → 37 hona chahiye, diya 38 → **38**
5. ×2+1 → 31×2+1 = **63**

**Set B:**
6. x=2,3; y=4,5 → **x < y**
7. x=3,4; y=2,3 → **x ≥ y**
8. x=2,2; y=2,2 → **x = y**
9. x=2,3; y=1,4 → **cannot be determined**
10. x=1,2; y=3,4 → **x < y**
