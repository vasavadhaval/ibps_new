# 🟡 Quant — Percentage (Basic) | Day 10

> Percentage = Week 3 ka doosra easy maths topic. SI ke saath milke cut-off clear karne me kaam aata hai.
> Sirf **basic** — complex success/failure chain abhi skip karo.

---

## 1. Core Formulas

```
x% of N = (x / 100) × N

N ka x% = (x/100) × N

Percentage change = (Change / Original) × 100
```

**Important conversions:**
- Fraction → %: 1/2 = 50%, 1/4 = 25%, 1/5 = 20%, 1/8 = 12.5%, 1/10 = 10%
- % → Fraction: 25% = 1/4, 20% = 1/5

---

## 2. "More than" / "Less than"

| Statement | Formula |
|---|---|
| A is **x% more** than B | A = B × (1 + x/100) |
| A is **x% less** than B | A = B × (1 − x/100) |
| Price increased by x% | New = Old × (1 + x/100) |
| Price decreased by x% | New = Old × (1 − x/100) |

---

## 3. Solved Examples

**Example 1:** 200 ka 15% = ?
```
(15/100) × 200 = 30 ✅
```

**Example 2:** 80 is what % of 200?
```
(80/200) × 100 = 40% ✅
```

**Example 3:** A number is increased by 20%. Original = 500. New value?
```
500 × (1 + 20/100) = 500 × 1.2 = 600 ✅
```

**Example 4:** A is 25% more than B. If B = 400, A = ?
```
A = 400 × (1 + 25/100) = 400 × 1.25 = 500 ✅
```

**Example 5:** Price 800 reduced by 10%. New price?
```
800 × (1 − 10/100) = 800 × 0.9 = 720 ✅
```

---

## 4. Quick Tricks

- **10%** = number ÷ 10
- **5%** = 10% ÷ 2
- **20%** = number ÷ 5
- **25%** = number ÷ 4
- **50%** = number ÷ 2

---

## 5. ✍️ Practice Questions

1. Find 20% of 450.
2. Find 15% of 800.
3. 60 is what % of 240?
4. A number increased by 25%. Original = 400. New value?
5. Price 1000 decreased by 10%. New price?
6. A is 20% more than B. B = 250. Find A.
7. 33.33% of 600 = ?
8. If 40% of a number is 80, find the number.
9. 150 increased by 20% = ?
10. Which is greater: 25% of 200 or 20% of 250?

---

## ✅ Answers

1. (20/100)×450 = **90**
2. (15/100)×800 = **120**
3. (60/240)×100 = **25%**
4. 400×1.25 = **500**
5. 1000×0.9 = **900**
6. 250×1.2 = **300**
7. (1/3)×600 = **200**
8. 80 = (40/100)×N → N = **200**
9. 150×1.2 = **180**
10. 25% of 200 = 50; 20% of 250 = 50 → **Equal**

---

> **Aaj ka takeaway:** x% of N = (x/100)×N. "x% more" = ×(1+x/100). Fraction-percentage conversions yaad rakho — speed ke liye. Roz 5 percentage questions karo.
