# 🟡 Quant — Profit & Loss (Basic) | Day 13

> Profit & Loss = CP (Cost Price), SP (Selling Price), Profit, Loss.
> Week 4 easy maths — sirf **basic** formulas. Heavy discount chain abhi skip karo.

---

## 1. Core Formulas

```
Profit = SP − CP        (jab SP > CP)
Loss = CP − SP          (jab CP > SP)

Profit % = (Profit / CP) × 100
Loss % = (Loss / CP) × 100

SP = CP × (1 + Profit%/100)   [profit par]
SP = CP × (1 − Loss%/100)       [loss par]
```

> **Yaad rakho:** Profit% aur Loss% hamesha **CP** par calculate hote hain, SP par nahi.

---

## 2. Solved Examples

**Example 1:** CP = 500, SP = 600. Profit?
```
Profit = 600 − 500 = 100
Profit % = (100/500) × 100 = 20% ✅
```

**Example 2:** CP = 800, Loss = 10%. SP?
```
SP = 800 × (1 − 10/100) = 800 × 0.9 = 720 ✅
```

**Example 3:** SP = 450, Profit = 12.5%. CP?
```
SP = CP × 1.125
CP = 450 / 1.125 = 400 ✅
```

**Example 4:** CP = 200, SP = 250. Profit %?
```
Profit = 50
Profit % = (50/200) × 100 = 25% ✅
```

**Example 5:** A man bought at Rs 400 and sold at Rs 350. Loss %?
```
Loss = 50
Loss % = (50/400) × 100 = 12.5% ✅
```

## 3. Quick Tricks

| Profit % | SP formula |
|---|---|
| 10% | SP = CP × 1.1 |
| 20% | SP = CP × 1.2 |
| 25% | SP = CP × 1.25 |
| 50% | SP = CP × 1.5 |

| Loss % | SP formula |
|---|---|
| 10% | SP = CP × 0.9 |
| 20% | SP = CP × 0.8 |
| 25% | SP = CP × 0.75 |

2. Loss = **60**, Loss % = **10%**
3. SP = 800 × 1.25 = **1000**
4. SP = 1000 × 0.9 = **900**
5. CP = 660/1.1 = **600**
6. CP = 360/0.9 = **400**
7. Profit = 100, % = (100/500)×100 = **20%**
8. Profit = 60, % = (60/240)×100 = **25%**
9. CP = 450/1.25 = **360**
10. SP = 500 × 0.8 = **400**
