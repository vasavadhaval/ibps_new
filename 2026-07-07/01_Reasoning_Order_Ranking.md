# 🧠 Reasoning — Order & Ranking | Day 14

> "Kaun sabse lamba?", "A se kitne log chhote hain?" — aise questions.

---

## 1. Common Question Types

| Type | Example |
|---|---|
| **Total persons + position from ends** | 40 students, Ram 15th from top — rank from bottom? |
| **Interchange position** | Ram 12th from top, 28th from bottom — total? |
| **Comparison** | A taller than B, B taller than C — who shortest? |
| **Height/Age order** | Arrange shortest to tallest |

---

## 2. Key Formulas

```
Total persons = Position from top + Position from bottom − 1

Position from bottom = Total − Position from top + 1

Position from top = Total − Position from bottom + 1
```

**Example:** 30 students. Rank from top = 10.
```
From bottom = 30 − 10 + 1 = 21st ✅
Total check: 10 + 21 − 1 = 30 ✅
```

---

## 3. Solved Examples

**Example 1:** In a row of 50, A is 18th from left. Position from right?
```
From right = 50 − 18 + 1 = 33rd ✅
```

**Example 2:** Rahul is 12th from top and 28th from bottom. Total students?
```
Total = 12 + 28 − 1 = 39 ✅
```

**Example 3:** A is taller than B. C is shorter than B. Order (tallest first)?
```
A > B > C → Tallest = A ✅
```

**Example 4:** In a class of 40, if rank from top is same as rank from bottom, find rank.
```
Top + Bottom − 1 = Total
r + r − 1 = 40 → 2r = 41 → not integer (odd total case)
If total 41: 2r = 42 → r = 21 (middle position) ✅
```

## 4. ✍️ Practice Questions

1. In a row of 45, X is 20th from left. Position from right?
2. Rank 15th from top and 25th from bottom. Total persons?
3. In a queue of 60, position from left is 25. Position from right?
4. A > B > C > D. Who is 2nd tallest?
5. Total 35. Position from top = position from bottom. Find position?
6. 50 people. A is 10th from left, B is 15th from right. How many between A and B if not overlapping?
7. Ram taller than Shyam. Shyam taller than Mohan. Shortest?
8. Rank from top 8, total 30. Rank from bottom?

---

## ✅ Answers

1. 45 − 20 + 1 = **26th**
2. 15 + 25 − 1 = **39**
3. 60 − 25 + 1 = **36th**
4. **B** (A tallest, B 2nd)
5. 2r − 1 = 35 → r = **18th** (middle)
6. A at 10, B at 15 from right = 50−15+1 = 36. Between 10 and 36 = 36−10−1 = **25** (if same row left-right)
7. **Mohan**
8. 30 − 8 + 1 = **23rd**
