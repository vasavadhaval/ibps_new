# 📖 English — Para Jumbles | Day 14

> Para Jumbles = 4-5 sentences **uljhe hue** diye hote hain. Sahi **order** (sequence) batana hota hai.
> Logic: pronoun link, connectors (but, therefore), chronology, general → specific.

---

## 1. Solve Karne Ka Tarika

1. **Opening sentence** dhoondho — usually general statement, no "he/she/this/that" without reference.
2. **Closing sentence** — conclusion, "thus", "therefore", summary.
3. **Pairs** dhoondho — sentence jo logically ek dusre ke baad aaye.
4. Pronoun wala sentence uske **noun wale** sentence ke baad aata hai.
5. Options me eliminate karo.

---

## 2. Clue Words

| Clue | Matlab |
|---|---|
| **A, An, The** (without prior noun) | Opening ho sakta hai |
| **He, She, It, They, This, That** | Koi pehle sentence chahiye |
| **But, However, Yet** | Pehle kuch contrast hona chahiye |
| **Therefore, Thus, Hence** | Closing / result sentence |
| **Also, Moreover, Furthermore** | Pehle related sentence hona chahiye |

---

## 3. Solved Example

**Sentences:**
- A) Therefore, regular practice is essential.
- B) Learning a language takes time and effort.
- C) Many people give up too early.
- D) However, consistency brings results.

**Order:** B (general opening) → C (problem) → D (contrast/turn) → A (conclusion)
**Answer: B-C-D-A** ✅

## 6. ✍️ Practice 3 (IBPS style — 5 sentences)

- P) Thus, banks must invest in cybersecurity.
- Q) Digital banking has grown rapidly in India.
- R) With growth, cyber threats have also increased.
- S) Customers now prefer online transactions.
- T) Hackers target weak security systems.

**Correct order?**

---

## ✅ Answers

**Practice 1:** **B-D-C-A**
- B opens (general). D (people ignore). C (doctors recommend). A (also benefits).

**Practice 2:** **A-S-C-B-D** or **A-C-S-B-D**
- A opens. S/C explain growth. B remote jobs. D challenges.
- Best: **A-S-B-C-D** — A (internet changed work) → S (online preference) → B (remote common) → C (pandemic shift) → D (challenges)

**Practice 3:** **Q-S-R-T-P**
- Q (digital banking grew) → S (customers prefer online) → R (threats increased) → T (hackers target) → P (Thus, invest in security)
