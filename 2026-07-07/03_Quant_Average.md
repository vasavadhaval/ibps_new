# 🟡 Quant — Average | Day 14

> Average = sabhi values ka **mean**. Easy topic — formula yaad rakho.

---

## 1. Core Formulas

```
Average = Sum of all values / Number of values

Sum = Average × Number of values

New average jab ek value add/remove ho:
New Avg = (Old Sum ± value) / (Old count ± 1)
```

**Weighted Average:**
```
Weighted Avg = (w1×x1 + w2×x2 + ...) / (w1 + w2 + ...)
```

---

## 2. Solved Examples

**Example 1:** Average of 5 numbers is 20. Sum?
```
Sum = 20 × 5 = 100 ✅
```

**Example 2:** Average of 10, 20, 30, 40, 50?
```
(10+20+30+40+50)/5 = 150/5 = 30 ✅
```

**Example 3:** Average of 5 numbers is 18. One number 12 replaced by 24. New average?
```
Old sum = 18×5 = 90. New sum = 90 − 12 + 24 = 102. New avg = 102/5 = 20.4 ✅
```

**Example 4:** Class avg 40 marks. 10 students avg 50. Class has 30 students. Remaining 20 students avg?
```
Total = 30×40 = 1200. First 10 = 500. Remaining sum = 700. Avg = 700/20 = 35 ✅
```
