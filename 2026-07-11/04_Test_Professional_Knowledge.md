1. Which is a DDL command?
   - a) SELECT  b) INSERT  c) CREATE  d) UPDATE

2. Primary Key can have:
   - a) Duplicate  b) NULL  c) Unique non-null  d) Multiple per table

3. 1NF ensures:
   - a) No duplicate rows  b) Atomic values  c) No NULL  d) No FK

4. 3NF eliminates:
   - a) Partial dependency  b) Transitive dependency  c) Redundancy only  d) NULL

5. Foreign Key references:
   - a) Same table  b) PK of another table  c) Any column  d) Index

6. ACID — Atomicity means:
   - a) Permanent  b) All or nothing  c) No interference  d) Valid data

7. Which clause filters rows?
   - a) SELECT  b) WHERE  c) ORDER BY  d) GROUP BY

8. INNER JOIN returns:
   - a) All left rows  b) Matching rows in both tables  c) All right rows  d) All rows

9. HAVING is used to filter:
   - a) Rows  b) Groups  c) Columns  d) Tables

10. A program in execution is called:
    - a) Thread  b) Process  c) File  d) Kernel

11. Which is NOT a process state?
    - a) Ready  b) Running  c) Compiled  d) Waiting

12. FCFS scheduling means:
    - a) Shortest first  b) First come first served  c) Priority  d) Round robin

13. Deadlock occurs when:
    - a) CPU idle  b) Processes wait for each other  c) Memory full  d) One process runs

14. Kernel is:
    - a) User app  b) Core of OS  c) Hardware  d) Browser

15. Virtual memory uses:
    - a) RAM only  b) Disk to extend RAM  c) Cache  d) ROM

7. **b) WHERE**

9. **b) Groups**
10. **b) Process**
