# 🟡 Quant — Ratio & Proportion | Day 20

> Ratio & Proportion = easy scoring topic. Cut-off ke liye useful.
> Formula clear rakho — calculator nahi milta.

---

## 1. Core Formulas

```
Ratio a : b = a/b

If a : b = m : n, then a = mk, b = nk (k = common multiple)

Proportion: a/b = c/d  →  a×d = b×c  (cross multiply)

a : b = c : d  means a/b = c/d
```

**Compound ratio:** a:b and c:d → (a×c) : (b×d)

**Divide in ratio:** Total = parts sum. Share = (part/total) × amount.

---

## 2. Solved Examples

**Example 1:** Ratio of A:B = 3:5. If A = 15, B = ?
```
3/5 = 15/B → B = 15×5/3 = 25 ✅
```

**Example 2:** Divide Rs 800 in ratio 3:5.
```
Parts = 3+5 = 8. One part = 800/8 = 100.
First = 3×100 = 300, Second = 5×100 = 500 ✅
```

**Example 3:** If 4 : 5 = x : 20, find x.
```
4/5 = x/20 → x = 4×20/5 = 16 ✅
```

**Example 4:** A:B = 2:3, B:C = 4:5. Find A:B:C.
```
A:B = 2:3 = 8:12
B:C = 4:5 = 12:15
A:B:C = 8:12:15 ✅
```

**Example 5:** Ages of A and B in ratio 4:5. Sum of ages = 45. Ages?
```
4+5=9 parts. One part = 45/9 = 5.
A = 20, B = 25 ✅
```

## 3. ✍️ Practice Questions

1. A:B = 2:5. If A = 10, B = ?
2. Divide Rs 600 in ratio 2:3.
3. If 3:4 = x:12, find x.
4. A:B = 3:4, B:C = 5:6. Find A:B:C.
5. Ratio of boys:girls = 5:3. Total 40. Girls = ?
6. Ages A:B = 5:7. Sum = 48. Age of A?
7. If 2:5 = 8:x, find x.
8. Rs 900 divided in 4:5. Larger share?
9. A:B = 1:2, B:C = 3:4. A:C = ?
10. Ratio 7:9. Difference of parts = 8. Smaller number?

---

## ✅ Answers

1. B = 10×5/2 = **25**
2. Parts 5, one=120 → **240 and 360**
3. x = 3×12/4 = **9**
4. A:B=3:4=15:20, B:C=5:6=20:24 → **15:20:24**
5. Girls = 3/8 × 40 = **15**
6. A = 5/12 × 48 = **20**
7. x = 5×8/2 = **20**
8. Larger = 5/9 × 900 = **500**
9. A:B=1:2=3:6, B:C=3:4=6:8 → A:C = **3:8**
10. Diff of parts = 2, each part = 4. Smaller = 7×4 = **28**
