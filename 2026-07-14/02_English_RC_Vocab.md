## Passage

Cloud computing allows users to store data and run applications on remote servers instead of personal computers. Banks use cloud services to scale systems quickly during high demand. Advantages include lower hardware cost and easy access from anywhere. Concerns include data privacy and dependence on internet connectivity. Experts recommend using hybrid models — sensitive data on private servers, other services on public cloud — to balance cost and security.

---

1. Main idea?
   - a) Cloud is useless  b) Cloud has benefits and concerns for banks  c) Internet is not needed  d) All data must be public

2. Advantage mentioned?
   - a) Higher hardware cost  b) Easy access / lower cost  c) More delay always  d) No security

3. Hybrid model means?
   - a) Only public cloud  b) Only private servers  c) Mix of private + public  d) No cloud

4. Concern mentioned?
   - a) Data privacy  b) Free lunch  c) Too many branches  d) Extra holidays

5. Synonym of **scale** (here): grow/adjust capacity — a) shrink only b) expand/adjust capacity c) delete d) print
