## 1. Quick Revision Sheet

### DBMS
- PK = unique, not null | FK = references other table PK
- 1NF = atomic | 2NF = no partial dep | 3NF = no transitive dep
- ACID = Atomicity, Consistency, Isolation, Durability
- DDL: CREATE, ALTER, DROP | DML: SELECT, INSERT, UPDATE, DELETE

### SQL
- WHERE = row filter | HAVING = group filter
- INNER JOIN = matching both | ORDER BY = sort
- Execution: FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY

### OS
- Process = running program | Thread shares memory
- States: Ready, Running, Waiting, Terminated
- FCFS, SJF, Round Robin (time quantum), Priority
- RAM = volatile | Virtual memory = disk | Page fault = load from disk

### Networks
- OSI 7 layers (Application → Physical)
- TCP = reliable | UDP = connectionless
- Router = Network layer | Switch = Data Link
- HTTP 80 | HTTPS 443 | DNS 53 | FTP 21 | SSH 22
- IPv4 = 32-bit | Class C = 192–223 | 192.168.x.x = private

## 2. ✍️ Mixed MCQ (15 Q · 15 min)

1. Which is DML? a) CREATE b) SELECT c) DROP d) ALTER

3. ACID Atomicity? a) All or nothing b) Permanent c) Valid d) Isolated
4. HAVING filters? a) Rows b) Groups c) Columns d) Tables
5. Process in execution = ? a) File b) Thread c) Process d) Kernel
6. Round Robin uses? a) Priority b) Time quantum c) Shortest job d) Random
7. Page fault means? a) In RAM b) Load from disk c) Delete d) Print
8. OSI layers count? a) 4 b) 5 c) 7 d) 8
9. TCP is at? a) Network b) Transport c) Physical d) Application
10. Router layer? a) Physical b) Data Link c) Network d) Transport
11. HTTPS port? a) 80 b) 443 c) 21 d) 53
12. DNS converts? a) IP to MAC b) Domain to IP c) File to packet d) Port
13. Class C range? a) 1–126 b) 128–191 c) 192–223 d) 224–239
14. Star topology = ? a) Bus b) Central hub c) Ring only d) Mesh only
15. UDP is? a) Connection-oriented b) Connectionless c) Always reliable d) Slowest

## ✅ Answers

2. **b) Transitive dependency**
3. **a) All or nothing**
4. **b) Groups**
5. **c) Process**
6. **b) Time quantum**
7. **b) Load from disk**
8. **c) 7**
9. **b) Transport**
10. **c) Network**
11. **b) 443**
12. **b) Domain to IP**
13. **c) 192–223**
14. **b) Central hub (star)**
