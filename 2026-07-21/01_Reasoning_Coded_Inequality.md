# 🧠 Reasoning — Coded Inequality | Day 28

> Week 6 topic. Pehle symbols decode → phir normal inequality jaisa solve.

---

## 1. How to solve (2 min)

1. Code table likho (P @ Q means P > Q, etc.).
2. Statement ko normal `>`, `<`, `≥`, `≤`, `=` me convert karo.
3. Conclusion check: Follows / Does not follow.

**Common codes (aaj ke sets):**

| Symbol | Meaning |
|---|---|
| `@` | > |
| `#` | < |
| `$` | ≥ |
| `%` | ≤ |
| `&` | = |

---

## 2. ✍️ Practice Set (10 Q · 25 min)

**Statements use codes above.**

1. `A @ B # C`  
   Conclusion: A > C — Follows?

2. `P $ Q & R`  
   Conclusion: P ≥ R — Follows?

3. `M % N @ O`  
   Conclusion: M < O — Follows?

4. `X & Y $ Z`  
   Conclusion: X ≥ Z — Follows?

5. `A @ B $ C # D`  
   Conclusion: A > D — Follows?

6. `P # Q % R & S`  
   Conclusion: P < S — Follows?

7. `A $ B # C @ D`  
   Conclusion: A > D — Follows?

8. `M @ N & O % P`  
   Conclusion: M > P — Follows?

9. `W % X $ Y # Z`  
   Conclusion: W < Z — Follows?

10. `A & B @ C # D`  
    Conclusion: A > D — Follows?

---

## ✅ Answers

1. A > B < C → A and C relation clear nahi → **Does not follow**
2. P ≥ Q = R → P ≥ R → **Follows**
3. M ≤ N > O → M and O clear nahi → **Does not follow**
4. X = Y ≥ Z → X ≥ Z → **Follows**
5. A > B ≥ C < D → A ? D unclear → **Does not follow**
6. P < Q ≤ R = S → P < S → **Follows**
7. A ≥ B < C > D → A ? D unclear → **Does not follow**
8. M > N = O ≤ P → M ? P unclear → **Does not follow**
9. W ≤ X ≥ Y < Z → W ? Z unclear → **Does not follow**
10. A = B > C < D → A ? D unclear → **Does not follow**

---

## Quick Rule
- Chain me **opposite direction** break → conclusion usually **Does not follow**.
- Same direction + `=` → often **Follows**.

---

> **Aaj ka takeaway:** Coded inequality = decode table + normal inequality. Unclear relation = Does not follow. Guess mat karo (−0.25).
