## 1. Notes (padho · 20 min)

### CIA Triad
| Goal | Meaning |
|---|---|
| **Confidentiality** | Sirf authorized access (encryption, passwords) |
| **Integrity** | Data change na ho unauthorized (hash, checksum) |
| **Availability** | System/service available (backup, uptime) |

### Common Attacks
| Attack | Simple meaning |
|---|---|
| **Phishing** | Fake mail/link se password/OTP chori |
| **Malware** | Virus, worm, trojan, ransomware |
| **Ransomware** | Files lock → money maangta hai |
| **DoS / DDoS** | Server overload → service band |
| **Man-in-the-Middle** | Beech me traffic sunna/change |
| **SQL Injection** | Form me malicious SQL |
| **Brute force** | Password guess repeatedly |

### Defenses
| Tool/Idea | Kaam |
|---|---|
| **Firewall** | Allow/block network traffic |
| **Antivirus** | Detect/remove malware |
| **Encryption** | Data ko code me badalna (HTTPS, AES) |
| **Authentication** | Who you are (password, OTP, biometrics) |
| **Authorization** | What you can access (roles/permissions) |
| **2FA / MFA** | Extra verification step |
| **VPN** | Secure tunnel over public network |
| **Patch/Update** | Security holes band karna |

### Banking terms
- **OTP** — kabhi share mat karo
- **HTTPS** — padlock, encrypted site
- **Social engineering** — logon ko trick karke info lena
