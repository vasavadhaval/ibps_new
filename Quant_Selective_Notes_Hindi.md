## 1. Simplification / Approximation (SABSE EASY ⭐)

**BODMAS order:** **B**rackets → **O**rder(power) → **D**ivision → **M**ultiplication → **A**ddition → **S**ubtraction.

**Approximation:** numbers ko nearest round figure me badlo.
- Example: `19.8 × 5.1 ≈ 20 × 5 = 100`.

**Yaad karne layak (speed ke liye):**
- Squares 1-30, Cubes 1-15.
- Tables 1-20.
- Fraction → percentage: 1/2=50%, 1/3=33.3%, 1/4=25%, 1/5=20%, 1/8=12.5%, 1/6=16.6%.

---

## 2. Number Series

Pattern dhoondho — kaun sa rule lag raha hai:
- **+/− pattern:** 2, 5, 8, 11... (+3)
- **×/÷ pattern:** 3, 6, 12, 24... (×2)
- **Square/Cube:** 1, 4, 9, 16... (n²)
- **Mixed:** difference ka difference dekho (2, 3, 6, 11, 18 → +1, +3, +5, +7)
- **Prime numbers:** 2, 3, 5, 7, 11, 13...

**Trick:** Consecutive numbers ke beech ka **difference nikaalo**, fir difference me pattern dhoondho.

## 3. Quadratic Equation (fast scoring)

Do equations di hoti hain (x aur y), aapko relation batana hai (x>y, x<y, x≥y, x≤y, ya relation nahi).

**Form:** `ax² + bx + c = 0`. Roots nikalne ke liye factorize karo.

**Shortcut (sign rule):**
- `x² − (sum)x + (product) = 0` → dono roots ka sign **product/sum** se decide.
- Agar `+bx +c` → dono roots **negative**.
- Agar `−bx +c` → dono roots **positive**.
- Agar `c` negative → ek positive, ek negative root.

**Compare:** dono equations ke roots nikaalo, fir har x ko har y se compare karke conclusion banao.

## 4. Simple Interest (SI)

```
SI = (P × R × T) / 100
```
- P = Principal (mool dhan), R = Rate (%), T = Time (saal).
- Amount = P + SI.

Example: P=5000, R=10%, T=2 saal → SI = (5000×10×2)/100 = **1000**.

---

## 5. Percentage (basic)

```
x% of N = (x/100) × N
```
- "A is x% more than B" → A = B × (1 + x/100).
- "A is x% less than B" → A = B × (1 − x/100).

Example: 200 ka 15% = (15/100)×200 = **30**.

## 6. Data Interpretation (sirf easy — Table / Bar Graph)

- Diye gaye data se total, average, percentage, ratio nikalna hota hai.
- **Pehle easy questions** (direct value, total) karo.
- Ratio/percentage wale aage. Complex caselet **chhod do**.
- Calculator nahi milta — isliye **approximation** use karo jahan options door-door ho.
