## 2. Sectional Cut-off — SABSE ZAROORI BAAT

IBPS SO Prelims me **har section ka alag cut-off** hota hai. Iska matlab:

> Sirf total 55 marks laana **kaafi nahi**. Aapko **teeno sections ka minimum cut-off** alag-alag clear karna hoga — **Maths bhi**.

Isliye "Maths bilkul nahi karunga" possible nahi hai. Lekin tension mat lo —
**Maths me sirf 6-8 easy questions** sahi kar lo toh cut-off aaram se nikal jaata hai.
Heavy calculation wale topics (Time-Work, Boat-Stream, complex DI) hum **chhod denge**.
