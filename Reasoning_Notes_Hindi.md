## 1. Inequality (असमानता) — Easy Marks ⭐

**Symbols:**
- `A > B` → A, B se bada
- `A < B` → A, B se chhota
- `A ≥ B` → A, B se bada ya barabar
- `A = B` → barabar

**Rule (combine karna):** Inequality tabhi combine hoti hai jab common element ke through ek hi direction ho.
- `A > B > C` → `A > C` ✅ (valid)
- `A > B < C` → A aur C ka relation **nahi** nikal sakte ❌

**Either-Or case:** Jab dono conclusions individually false ho lekin ek common variable par opposite signs (>, <, =) cover karein.
- Example: `A ≥ B` aur `B = C` nahi diya, sirf `A ≥ B` ho. Conclusions: `A > B` aur `A = B`. Dono alag-alag false, par milke poora cover karte hain → **Either-Or true**.

**Trick:** Pehle saare symbols ko ek line me likho, fir conclusion check karo.

## 2. Syllogism (न्यायवाक्य) — Venn Diagram se solve

**Statements ke types:**
- "All A are B" → saara A, B me hai
- "Some A are B" → kuch A, B me hai
- "No A is B" → A aur B alag

**Golden Rules:**
1. `All + All = All` (A→B, B→C ⇒ All A are C)
2. `All + No = No`
3. `Some + All = Some`
4. `Some + No = Some not`
5. `Some` ka reverse bhi `Some` hota hai (Some A are B ⇒ Some B are A) ✅
6. `All A are B` ⇒ `Some B are A` (definitely true)

**Possibility:** "Some A are B" diya ho toh "All A are B" ek **possibility** ho sakti hai.

**Trick:** Hamesha **Venn diagram** banao, sabse minimum overlap maan ke check karo.

## 3. Coding-Decoding (कूट भाषा)

**Common types:**
- **Letter shifting:** CAT → DBU (har letter +1). Reverse ke liye -1.
- **Number coding:** A=1, B=2 ... Z=26 (ya reverse Z=1).
- **New pattern / symbol coding:** word ko digit + symbol diya jaata hai, pattern dhoondho.

**Tip:** Alphabet positions yaad rakho:
```
A=1  B=2  C=3  D=4  E=5  F=6  G=7  H=8  I=9  J=10
K=11 L=12 M=13 N=14 O=15 P=16 Q=17 R=18 S=19 T=20
U=21 V=22 W=23 X=24 Y=25 Z=26
```
**Reverse trick:** Reverse position = `27 − normal position`. (A↔Z, B↔Y...)

## 4. Alphanumeric Series — Easy ⭐

Ek series di hoti hai (letters + numbers + symbols mix). Questions: "kitne number ke baad symbol", "left/right se nth element" waghaira.

**Approach:**
- Series ko dhyaan se likho, positions left & right dono se gino.
- "Left se 5th ke right ka 3rd" = left se (5+3=8)th element.

**Trick (X se left + Y se right):** Agar total N elements, "left se Pth = right se (N−P+1)th".

## 5. Direction & Distance (दिशा)

```
        North
          |
West ---- + ---- East
          |
        South
```
- **Right turn** = clockwise (N→E→S→W).
- **Left turn** = anti-clockwise (N→W→S→E).
- **Distance:** seedhe movement add/subtract karo. Diagonal ke liye **Pythagoras**: `√(a² + b²)`.

**Pythagoras yaad rakho (common triplets):** (3,4,5), (5,12,13), (6,8,10), (8,15,17).

## 6. Blood Relation (रक्त संबंध)

**Symbols use karo (family tree banao):**
- `+` = Male, `−` = Female
- `→` ya lines se generation dikhao.

**Common relations:**
- Father's/Mother's son = Brother
- Father's brother = Uncle (Chacha/Mama)
- Son's wife = Daughter-in-law (Bahu)

**Coded blood relation:** `A + B` = A is father of B type. Diye gaye symbols ke hisaab se tree banao.

**Trick:** "Only son/daughter" jaise words dhyaan se padho. Generation ka level fix karo.

## 7. Order & Ranking (क्रम)

- "A is 5th from top, 8th from bottom" ⇒ **Total = 5 + 8 − 1 = 12**.
- Formula: `Total = (rank from top) + (rank from bottom) − 1`.

## 8. Puzzles & Seating Arrangement — SABSE ZYADA MARKS 🔥

Ye section sabse zyada (15-20 marks). Practice se hi master banoge.

### Types:
1. **Linear arrangement** — ek line me log (north/south facing).
2. **Circular arrangement** — gol me (center ki taraf / bahar ki taraf mooh).
3. **Box/Floor puzzle** — floors/boxes me cheezein arrange.
4. **Month/Day based** — alag-alag din/mahine me events.

### Solving Method (step-by-step):
1. **Direct hints** (jo pakka fix karte hain) pehle lagao.
2. **Possible cases** banao jab confusion ho (Case 1, Case 2).
3. Negative hints (jaise "A, B ke saath nahi baithta") se cases eliminate karo.
4. Jo case contradict kare use **kaat do**.

### Facing rule (circular):
- **Center ki taraf mooh:** left = clockwise, right = anti-clockwise.
- **Bahar ki taraf mooh:** ulta ho jaata hai.

> **Tip:** Roz **2-3 puzzles** solve karo. Pehle time mat dekho, accuracy banao; baad me speed apne aap aayegi.

## 9. Data Sufficiency

Question ke saath 2 statements (I & II) diye hote hain. Aapko batana hota hai **kaun sa statement answer ke liye kaafi hai** — answer nikalna **zaroori nahi**.

**Standard options:**
- (a) Sirf I kaafi hai
- (b) Sirf II kaafi hai
- (c) Dono milke kaafi hain
- (d) Dono alag-alag kaafi hain
- (e) Dono milke bhi kaafi nahi

**Trick:** Har statement ko **alag se** check karo, dono ko ek saath answer dhoondhne ke lilalach me mat phaso.

## General Reasoning Tips
- Pehle **easy topics** (Inequality, Syllogism, Series, Direction) attempt karo — guaranteed marks.
- Puzzle me **2 min** se zyada atak jao toh chhod ke aage badho, baad me aao.
- Negative marking se bachne ke liye **guess mat karo**.
- Roz **timed practice** — yahi reasoning me sabse zyada sudhar laata hai.
